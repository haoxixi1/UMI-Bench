# UMI-Bench 1.0 Project Page

Static project page for **UMI-Bench 1.0: An Open and Reproducible Real-World Benchmark for Tabletop Robotic Manipulation with UMI Data**.

## Local Preview

```bash
python3 -m http.server 8008
```

Then open:

```text
http://localhost:8008
```

## GitHub Pages

This repository is ready to publish from the repository root.

In GitHub:

```text
Settings -> Pages -> Build and deployment -> Deploy from a branch -> main / root
```

For a project repository, the page will be available at:

```text
https://<username>.github.io/<repository>/
```

For a user site repository named `<username>.github.io`, it will be available at:

```text
https://<username>.github.io/
```
