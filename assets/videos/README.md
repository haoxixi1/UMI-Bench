# Video Asset Naming

The homepage expects short, web-optimized MP4 files at the following paths.

## Top-Level Videos

- `hero-montage.mp4`: many-data/many-task montage for the first viewport
- `overview.mp4`: paper overview video shown after the overview text; web version is H.264 MP4, 1280x720, with AAC audio
